# MockAuthPlayground
A mock authentication app built with Avalonia UI and C#. Not intended for real use. Passwords are base64 encoded. Built for learning more about Avalonia, file I/O, and MVVM.

## Features (soon enough)
- User registration and login
- Interaction between accounts
- Username and password changing

## Stack
- Avalonia UI
- CommunityToolkit.Mvvm
- .NET 10

## How to run
1) Clone the repository
2) Open in IDE/Terminal
3) Run `dotnet run`
